Reservas

Controla las reservas activas, finalizadas y canceladas. Revisa el historial, gestiona pagos y verifica el estado de cada operación.

  • Utiliza el buscador para encontrar clientes o vehículos rápidamente.
  • Revisa el historial de reservas para mantener el control de las operaciones.
  • Actualiza el estado de los vehículos para evitar errores en la gestión.
